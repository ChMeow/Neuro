# Neuro (The backpropagetion)
A Simple NeuroNet Project

Basic Backpropagation function. (Without momentum and bias, etc.)

To test it,
1. Extract the LOGIC GATE (SAMPLE DATA).zip
2. Select the folder and adjust the setting accordingly.
3. Weight will be automatically saved into WEIGHT folder after backpropagation.

NOTE FOR NEW DATA TEXT FILE: DO NOT ENTER A NEW LINE AFTER THE LAST DATA, JUST FOLLOW THE SAMPLE DATA.
